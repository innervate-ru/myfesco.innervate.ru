Генерация файла Excel

Дата последнего обновления: 2018-11-27
Раздел Экспорт данных терминального слежения позволяет сгенерировать и выгрузить в Excel данные по терминальному слежению.
Вы можете также создать шаблон выгрузки и использовать его.

Что такое терминальное слежение?
Терминальное слежение — это внутренний термин, обозначающий отслеживание и запись всех событий, происходящих в течение перевозки.

В терминальном слежении вы можете увидеть следующие события и их даты:
— досмотры,
— взвешивания,
— даты сдачи на ЖД,
— погрузку на судно и др.
Как сгенерировать файл Excel

а) Зайдите на страницу Экспорт данных терминального слежения и нажмите оранжевую кнопку “Выгрузить данные” ;


б) Заполните необходимые вам поля в фильтре, чтобы отфильтровать данные для выгрузки ;

— “Юридическое лицо”
Выборка по наименованию клиента в заявке, либо по всем клиентам привязанным к данному аккаунту;

— “Дата создания заявки”
Выборка по времени создания заявки (заявки созданные в заданный период);
Если поле “По” не указано, то по умолчанию его значением будет время на момент выполнения выгрузки;

— “Последнее обновление”
Выборка по дате последнего обновления данных;
Данные терминального слежения периодически обновляются — например, появляются даты наступивших событий, появляются данные о МИДК, номера коносаментов и др. Эта выборка нужна для фильтрации по этой дате изменения данных терминального слежения;

— “Автор”
Выборка по автору заявки;
Значение “Любой пользователь ЛК” - выборка будет произведена без зависимости от автора заявки.


в) Выберите галочками поля, которые вы хотите выгрузить в Excel ;


г) Нажмите оранжевую кнопку ;


д) После нажатия на кнопку происходит автоматическая переадресация на страницу со списком заданий на выгрузки в Excel.
Несколько секунд ваше задание будет находиться “в процессе” генерации, в таком состоянии — вы можете отменить задание.
Далее состояние сменится на выполненное, значит вы можете скачать файл Excel ;


е) В списке заданий находим своё “задание” (самый первый элемент сверху) — нажимаем кнопку “Открыть” и далее кнопку “Скачать”.


ж) После выполнения задания выгрузки на почту отправится письмо.
В случае успешного выполнения в письме будет отражена характерная информация, также и в случае ошибки.

Результаты выполнения заданий (файлы Excel) хранятся ограниченное время – после прохождения которого, данные удаляются.
При этом Задание остается в истории исполнения заданий со статусом “Файл удалён”.


Время хранения сформированного Excel-файла экспорта данных терминального слежения — не более 24 часов.
Мы ограничиваем время хранения файлов с целью повышения быстродействия личного кабинета.
Успейте скачать сформированный файл до его удаления, пожалуйста.
