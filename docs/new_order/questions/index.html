Популярные вопросы

Дата последнего обновления: 2018-10-31
Как добавить еще один груз в заявку?

Необходимо в поле ГНГ указать код для сборного груза: 99020000.
И в поле “Дополнительные требования” указать разбивку по грузам.

Где указывается информация по дверной доставке?

В поле “Дополнительные требования”.

Кого указывать получателем на станции назначения?

В случае вывоза FESCO, указывается “агент ФИТ”.

Что указывать в поле Номер заказа?

При заполнении данного поля в заявке в ЛК будет фигурировать Ваш внутренний номер заявки.
